### 问题1：
在复制运用百度云上提供的代码的时候，出现了 Python 文件编码不匹配 的问题。具体报错信息如下：
> **Non-UTF-8 code starting with '\xbb' in file c:/Users/Questiiing/Desktop/BA.py on line 10, but no encoding declared**

起初在CSDN上查找解决办法，采用的是在代码最开头加入 # coding=gbk ，以此来声明用的是什么编码，但效果不好，没有解决问题。
问了ai得到了更简单的方法：
==利用记事本打开python文件，然后另存为UTF-8编码的格式，以此更改编码格式成功解决了问题。==

### 问题3：
为了配合后端的使用，必须在代码中添加__init__.py空文件，有__init__.py的收获，目录才会被识别为包，可以通过 import 包名 或 from 包名 import 模块 导入其中的内容。
